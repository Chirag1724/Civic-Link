# Digital Complaint Tracker (Tailwind Scaffold)

This archive contains a working scaffold for the Digital Complaint Tracker:
- Backend: Node + Express + MongoDB (Mongoose)
- Frontend: React + Vite + Tailwind

How to run (locally or on Replit):

1. Backend
   - cd backend
   - .env already set to mongodb://localhost:27017/Digital_Complaint_Tracker
   - npm install
   - npm run dev

2. Frontend
   - cd frontend
   - npm install
   - (Optional) create `.env` with VITE_API_URL=http://localhost:5000/api
   - npm run dev

Notes:
- The DSA modules are implemented as simple illustrative modules (HashSet, PriorityQueue, Graph with Dijkstra).
- Worker assignment uses a mock distance heuristic for demonstration. Replace with geolocation-based logic for production.
